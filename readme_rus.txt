SokobanCompact v0.7
05.02.2013

Бесплатная (open source) логическая игра типа Sokoban для Windows Mobile 6 и совместимых.
Суть игры: необходимо расставить ящики по обозначенным местам, ящики можно только толкать и только по одному за раз.
(см. также http://ru.wikipedia.org/wiki/Sokoban)

Возможности SokobanCompact:
- Удобное управление с помощью стилуса/тачскрина - перемещение по уровню за 1 клик, перетаскивание ящика - за 2 клика
- Управление клавишами
- Масштабирование и прокрутка - можно играть на больших уровнях с нормальным масштабом
- Автоматическое масштабирование уровня по полезным клеткам (отключаемо)
- Отмена сделанных ходов и возврат - Undo/Redo
- Сохранение и загрузка позиций - можно сохранить не завершённое положение
- Анимация движения по уровню и воспроизведение решения (отключаемо)
- Индикация запрещённых клеток - куда нельзя ставить ящики (иначе ящик нельзя будет передвинуть ни на одну цель; отключаемо)
- Индикация, что игра зашла в тупик (т.е. что уровень не может быть решён из текущего состояния)
- Подсчёт статистики игры - количество шагов, толканий, смен ящика и т.д.
- Учёт пройденных уровней и сохранение решений каждого уровня
- Добавление уровней - в комплекте с игрой идёт 31 набор (в сумме около 1800 уровней), новые наборы могут быть добавлены в дальнейшем (поддерживается текстовый формат)
- Изменяемая графика (скины) - поддерживаются пользовательские скины в формате bmp и их наборы
- Поддерживаются уровни с неравным количеством ящиков и целей

Что может показаться недостатками:
- Нет инсталлятора, zip-архив необходимо вручную распаковать и загрузить в устройство
- Интерфейс на английском языке
- Рассчёт запрещённых клеток может быть довольно долгим для больших уровней (например, на Sasquatch6, 50й уровень - 35х35 клеток)
- Игра разработана в основном для QVGA (240x320), однако работает и при других разрешениях экрана

Требования:
- Windows Mobile 6.0 или совместимая (возможно, будет работать на Windows Mobile 5.0)
- .NET Compact Framework 2.0
Работает на ПК (под Windows XP и т.п.), но графика при этом отображается не совсем корректно.

Известные аналоги для Windows Mobile:
- SokoSave (нет ручного масштабирования, не показывает запрещённые клетки)
- PocketSokoban (платный, перетаскивание ящика за 2 клика только на небольшие расстояния)
- Resco Sokoban (нет управления с помощью стилуса)

О научной сложности игры:
Задача решения уровней Sokoban относится к классам сложности NP-трудных (NP-hard) и PSPACE-полных (PSPACE-complete).
(см. также http://en.wikipedia.org/wiki/Sokoban#Scientific_research_on_Sokoban )
